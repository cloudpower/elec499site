Problem

Control and monitor your connected electronics from anywhere.

The cloudpower system consists of a power-bar unit with electrical outlets and a remote service, allowing power usage logging and management of connected devices from anywhere. Usage data can be retrieved and graphed easily from a simple web application.

Try out the demo api now.
api.gocloudpowered.com

Solution

The cloudpower unit controls and monitors connected devices using an Arduino, relays and a RMS currentvoltage monitoring circuit.

A Raspberry Pi runs a custom node.js monitor application, establishing a bidirectional WebSocket connection between the remote node.js web server and the Arduino firmware. This setup permits operation behind proxies and routers.

The usage data is submitted to the remote web server and is stored in a database, retrievable via a web application and JSON API.
